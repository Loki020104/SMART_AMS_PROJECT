🚀 HOW TO IMPORT ON YOUR WEBSITE:
================================================================================

Step 1: Login as Admin
   → Navigate to: https://smart-ams-project-faa5f.web.app
   → Login with Admin credentials

Step 2: Go to User Management
   → Click: "Dashboard" → "User Management" (in sidebar)
   → OR navigate to: Admin Portal → User Management tab

Step 3: Bulk Import Students
   → Click: "📥 Bulk Import" tab
   → Click: "📁 Choose File" → select: students_1500_final.csv
   → Click: "🔄 Preview" to verify 1500 students loaded
   → Click: "✅ Import" button
   → Wait for: "⏳ Starting bulk import..."
   → Confirm: "✅ Success: 1500 users imported" message

Step 4: Bulk Import Faculty  
   → Same process with faculty_72_final.csv
   → Click: "📥 Bulk Import" tab
   → Click: "📁 Choose File" → select: faculty_72_final.csv
   → Click: "🔄 Preview" to verify 72 faculty loaded
   → Click: "✅ Import" button
   → Confirm: "✅ Success: 72 users imported" message
